hare_bot

Цей бот є помічником у систематизації контенту яким ми користуємось повсякденно.
Цей бот вміє:
    - сортувати файли за класами (музика, картинки, відео, текст, інші файли);
    - в обраних класах сортувати файли за розділами ("для роботи", "для розваги", "треба переслати", "щоб було");
    - показувати погоду за обраним містом;
    - відстежує повітряну тривогу за обраним містом;
    - має власну телефонну книгу.

Список команд:
help - опис бота, список та роз'ястнення команд
box - відкриває доступ до сховища файлів
save - зберегти файл
weather - показати погоду
alarm_on - відстежувати повітряну тривогу
alarm_off - не відстежувати повітряну тривогу
book - доступ до телефонної книги
add - додати запис
delete - "name" - видалити запис за іменем
stats - кількість записів
list - список всіх імен в книзі
show - "name" - детальна інформація за іменем


для збереження файлу просто відправ його боту, потім у зручний час
пропиши команду "save" -> обери тип файлу -> обери розділ де буде зберігатися файл
