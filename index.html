Цей бот створений для забезпечення автоматизованої комунікації з користувачем через месенджер Telegram. Ідея проекту полягає в створенні віртуального асистента, який здатен виконувати різноманітні завдання та надавати корисні функції.

Основний функціонал чат-боту включає:

Погоду на сьогодні: отримання інформації про погоду в певному місті.
Телефонну книгу: можливість додавати, переглядати та видаляти контакти.
Конвертор валют: здійснення конвертації між різними валютами.

Додаткові функції:

1. Отримання інформації про власний ID у телеграм.

2. Список усіх доступних коман.

3. Перехід на веб-сайт бота.

title

description

Список команд:

start - Запуск бота, привітання з користувачем

menu - Відкриває головне меню бота

site - Пропонує варіанти переходу на сайт боту

converter - Відкриває конвертор валют

phone_book_menu - Відкриває меню телефонної книги

add_contact - Додати контакт до телефонної книги

show_contacts - Показати всі контакти телефонної книги

delete_contact - Видалити контакт за іменем

commands - Показує список усіх команд

id - Дізнатися свій id у телеграмі
